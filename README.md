# SpeedKey - 视频倍速助手

一款强大的 Chrome 扩展，让您可以通过按住指定按键来以自定义倍速播放网页中的所有视频，松开按键后立即恢复正常速度。

## ✨ 核心功能

### 🎯 按键控制
- **按住触发**：检测到用户按下（并持续按住）设定的按键，将页面上所有视频的播放速度设为自定义倍速值
- **松开恢复**：检测到用户松开按键，立即将视频速度恢复到正常速度（1×）
- **断焦保护**：如果按键按住后切换标签页或窗口，自动恢复正常速度，避免"卡在倍速"状态

### ⚙️ 可配置选项
- **触发按键**：支持自定义触发键（Shift、Alt、Ctrl、空格键等）
- **倍速数值**：支持 0.5× 到 5.0× 的任意倍速设置
- **界面显示**：可选择是否显示倍速浮层提示

### 🎛️ 用户体验
- **浮层提示**：在页面右上角显示当前倍速数值，便于实时感知
- **滚轮微调**：按住触发键时，可用鼠标滚轮实时微调倍速（±0.1× 级别）
- **输入框友好**：在文本输入区域时自动忽略按键操作，不影响正常输入

### 🔧 兼容与健壮性
- **动态检测**：自动检测新加载的视频元素（通过 MutationObserver）
- **智能识别**：支持各种视频网站和动态加载的视频内容
- **性能优化**：轻量级设计，不影响页面正常浏览

## 🚀 安装使用

### 安装步骤
1. 下载或克隆此项目到本地
2. 打开 Chrome 浏览器，进入扩展管理页面（`chrome://extensions/`）
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择项目文件夹
5. 扩展安装完成，首次安装会自动打开设置页面

### 基本使用
1. 在任意包含视频的网页上
2. 按住您设定的触发键（默认：左 Shift）
3. 视频立即以设定倍速播放，浮层显示当前倍速
4. 松开按键，视频恢复正常速度

### 高级功能
- **实时微调**：按住触发键时使用鼠标滚轮上下滚动可微调倍速
- **快速设置**：点击扩展图标可快速调整常用设置
- **详细配置**：在选项页面可进行详细的个性化配置
